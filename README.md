# AdynCMS

Система администрирования "Динамика", для проектов веб-студии [Динамика](http://adyn.ru)

## Базовая CMS

Подходит для большинства корпоративных сайтов

## Возможности

* Страницы
* Управление меню
* Новости
* Фотогалереи
* Партнеры
* Услуги
* Редиректы
* Слайдер
* Обратная связь
* Пользователи
